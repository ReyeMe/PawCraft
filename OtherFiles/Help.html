Welcome to PawCraft

This document contains some basic help for this program.

3D view

Camera Movement:

Key Action
W Move forward
S Move backwards
A Step left
D Step right
Q Move up
E Move down
SHIFT When combined with other movement keys it makes the camera move faster

Visibility:

Key Action
L Toggle grid lines visibility
N Show tile normal vectors

Paint tool

Key Action
CTRL Hold to switch to rotation mode

Cursor tool

This tool does absolutely nothing.

Raise tool

This tool raises terrain within area defined by Radius field.
The maximum height limit of the affected terrain can be set by Max level field.

Dig tool

This tool lowers terrain within area defined by Radius field.

Paint tool

Using this tool you can paint and rotate texture on the terrain.
Area affected in paint mode can be changed by modifing Radius field. To Rotate a texture, hold CTRL key while cliking the tile using Left mouse button.
Textures can be selected either by choosing their name in combo box, or by clicking on their preview in the Paint tool window.

Light tool

Using this tool you can set environment light in the level.

Entities tool

Using this tool you can place different entities onto a map. Just select type you like and using Left mouse button click anywhere on the map.
To delete already placed entity, select pointer tool, click using the Left mouse button on the entity you want gone and pres DEL key.
